EphemeralREST
-------------

EphemeralREST is a REST API that enables developers of astronomy and astrology applications to 
perform the sometimes complex calculations with simple calls to a REST service.

This application is still at pre-release though it is envisaged that it will become tested
and stable for production use within the next few months, as the developer is relying on 
it for accurate calculations himself.

This service relies on Swiss Ephemeris and it's data files which are also licenced under
the AGPL license. You may also obtain a commercial license to use Swiss Ephemeris from
Astrodienst.
